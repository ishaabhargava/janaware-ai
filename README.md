# EduFlow-AI

## Overview
EduFlow-AI is a web-based learning assistant that helps users practice quizzes, receive instant feedback, and improve understanding through guided hints.  
The system dynamically generates questions, evaluates responses, and adapts to user performance.

---

## Features
- Quiz generation from question database  
- Real-time answer evaluation  
- Hint system for guided learning  
- Adaptive difficulty (based on performance)  
- Simple and responsive UI
